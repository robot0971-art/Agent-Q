## 🚀 향후 과제 (Next Steps)

### 1. 세션 관리 및 대화 저장 (우선순위: P1)
현재 CLI 종료 시 사라지는 대화 기록을 영구적으로 관리한다.
- `/save <file>`: 현재 대화 내역(`ChatConversationHistory`)을 JSON 파일로 저장.
- `/load <file>`: 이전 대화 파일을 불러와 세션 복구.
- 종료 시 자동 저장 옵션 검토.

### 2. 설정 파일 기반 영속성 (우선순위: P1)
매번 API 키나 모델명을 입력하지 않도록 로컬 설정을 관리한다.
- `config.json` 파일을 통한 프로바이더, 모델, 베이스 URL 정보 저장 및 로드.
- `/config save`: 현재 설정을 기본값으로 저장하는 명령어 추가.

### 3. 안정성 및 보안 심화 (우선순위: P2)
- **재시도 로직:** API 호출 실패 시 지수 백오프(Exponential Backoff)를 적용한 Retry 로직 추가.
- **워크스페이스 루트 연동:** `AGENTQ_WORKSPACE_ROOT` 환경 변수를 `ToolPathGuard`와 완벽히 결합하여 보안 강제성 확보.

### 4. 사용자 경험(UX) 개선 (우선순위: P3)
- 도구 실행 중 진행률 표시(Spinner 등) 추가.
- 멀티라인 입력 지원 또는 긴 텍스트 입력 편의 기능.
